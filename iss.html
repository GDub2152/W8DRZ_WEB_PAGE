Beginner’s Guide to Working the ISS Crossband Repeater

Based on Bill Chaikin, KA8VIT’s tutorial.

What you need

About 25 watts ERP and a decent antenna. The ISS repeater uplink is on 2 meters and the downlink is on 70 cm, so downlink Doppler shift must be managed while listening.

The ISS repeater input requires a 67 Hz tone.

Four-step process

  1. Set up your radio memories.
  2. Check a tracking program for overhead passes.
  3. Check whether the ISS radio system is active.
  4. Make contacts using callsign and grid square.
