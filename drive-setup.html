Google Drive Library Setup

Use Google Workspace as the club document library while keeping the public website on GitHub Pages.

How it works

Club officers upload files into a shared Google Drive folder. A small Google Apps Script reads that folder and returns a clean JSON list. The Library page displays the files automatically.

  1. Create a shared Google Drive folder named TBOP Website Library.
  2. Add category folders such as Ham Radio, Antennas, Manuals, Meeting Minutes, Newsletters, and Photos.
  3. Copy the folder ID from the Google Drive URL.
  4. Paste the included google-apps-script-drive-library.gs into Google Apps Script and replace the folder ID.
  5. Deploy it as a Web App available to Anyone.
  6. Paste the Web App URL into assets/library-config.js.

Files included

assets/library-config.js
Where you paste the Web App URL.

assets/library-drive.js
Loads and displays Drive files on the Library page.

google-apps-script-drive-library.gs
The Google Apps Script code to install in Workspace.
